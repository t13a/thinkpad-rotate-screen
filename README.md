# ThinkPad Rotate Screen

Screen rotation script for ThinkPad.

## Prerequisites

- Xorg
    - `xinput` command
    - `xrandr` command
- Synaptics input driver
    - `synclient` command

## Basic usage

```sh
$ ./thinkpad-rotate-screen normal
$ ./thinkpad-rotate-screen left
$ ./thinkpad-rotate-screen right
$ ./thinkpad-rotate-screen invert
```
